Twitter automatic scroll

Twitter 자동으로 스크롤하기

아래 내용을 console에 붙여 넣자.

setInterval(function() { 
  if ($('.new-tweets-bar').length > 0) {
    $('.new-tweets-bar')[0].click();
  }
}, 1000);

이런거 chrome extension으로 만들어 놓으면 좋겠네.
